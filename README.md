# Airvd
### Basic CRUD REST demo using Django REST Framework.


### Used:
- Python
    - Django
    - Django REST Framework
    
### Run:
- Create `virtualenv` using `requirements.txt`
- Run `python manage.py runserver` in commandline. By default, server runs on port `5000`.
